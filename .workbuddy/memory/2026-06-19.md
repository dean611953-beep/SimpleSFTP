# 2026-06-19 工作日志

## SimpleFTP - Windows 轻量 FTP 服务端

**任务**: 从零开发一个 Windows 桌面工具，让 Windows 电脑变成 FTP 服务端，接收客户上传的文件。

### 经过

1. **初始需求误解**：一开始理解为 SFTP 客户端（从远程下载文件到本地）
   - 编写了完整的 SFTP 客户端架构
   - 推送了旧版本代码到 GitHub

2. **需求更正**：Dean 纠正——实际需求是 **本机作为 FTP 服务端**，客户通过 FTP 工具上传文件到本机
   - 推翻了原有代码
   - 全新重写为 FTP 服务端

3. **核心开发**：
   - 实现了完整的 FTP 协议支持（RFC 959）
   - 包括 PASV 模式、断点续传（REST）、目录操作（CWD/LIST/NLST）
   - 实现了上传文件记录、实时日志、配置持久化
   - WPF 界面：启动/停止按钮、状态面板、上传文件列表

4. **发布**：
   - GitHub: https://github.com/dean611953-beep/SimpleSFTP
   - 最后一次 commit: `7a08004 - Rewrite: FTP Server instead of SFTP Client`

### 技术要点

- 使用 .NET 原生 `TcpListener`，不依赖第三方 FTP 库
- 多线程并发处理多个客户端连接
- 被动模式（PASV）端口自动分配
- 上传文件按目录索引存储
- 配置保存至 `%APPDATA%\SimpleFTP\config.json`
