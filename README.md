# SimpleFTP v1.0.0 - Windows 轻量 FTP 服务端

> 让任何人的 Windows 电脑变成一个 FTP 接收服务器

## 如何使用

### 1. 下载
从 GitHub Releases 下载 `SimpleFTP-v1.0.0.zip`

### 2. 配置环境
安装 .NET 8.0 Runtime（如果还没装）:
https://dotnet.microsoft.com/download/dotnet/8.0

### 3. 运行
双击 `SimpleFTP.exe`

### 4. 设置

| 项目 | 默认值 | 说明 |
|------|--------|------|
| 端口 | 21 | FTP 服务端口 |
| 用户名 | admin | 客户登录用户名 |
| 密码 | - | 客户登录密码（必填） |
| 保存目录 | ~/Downloads/FTP_Uploads | 上传文件存放位置 |

### 5. 启动
点击 **▶ 启动** 按钮

### 6. 告诉客户

```
FTP 服务器: [你的局域网IP]
端口: 21
用户名: admin
密码: [你设置的密码]
```

## 功能

- **一键启停**：启动/停止 FTP 服务
- **自动记录**：每次上传的文件（文件名、大小、时间、IP）
- **实时监控**：上传日志实时更新
- **断点续传**：支持 FTP REST 指令
- **被动模式**：支持 PASV 模式
- **配置持久化**：端口、密码等自动保存

## 系统要求

- **操作系统**: Windows 10/11 (64位)
- **.NET Runtime**: .NET 8.0 Runtime

## 网络配置

### 局域网使用
直接启动即可，同一局域网内的设备通过你的 IP 连接

### 外网使用
需要路由器配置端口转发（端口 21 → 本机 IP）
