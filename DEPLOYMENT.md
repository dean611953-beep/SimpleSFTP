# SimpleFTP 部署指南

## 编译和运行

1. **安装 .NET 8.0 Runtime**: https://dotnet.microsoft.com/download/dotnet/8.0

2. **双击 `publish.bat`** 编译并发布

3. **运行 `publish/SimpleFTP.exe`**

## 首次使用

1. 设置端口（默认21）、用户名、密码
2. 选择文件保存目录
3. 点击"启动"
4. 把你的IP和账号信息发给客户

## 客户连接示例

**使用 WinSCP**:
- 主机: 你的IP
- 端口: 21
- 用户名: admin
- 密码: xxxxx
- 协议: SFTP → 改为 **FTP**

**使用 FileZilla**:
- 主机: 你的IP
- 端口: 21
- 用户名: admin
- 密码: xxxxx
- 加密: 只使用清传FTP
